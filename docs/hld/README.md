# High-Level Design Documents

This directory contains High-Level Design (HLD) documents for this project.
Each HLD captures the architecture, components, data flows, and key design decisions for a significant system, feature, or service.

To create a new HLD, copy [`HLD-00-template.md`](HLD-00-template.md), rename it following the `HLD-XX-<short-title>.md` convention, and fill in every section.

## Status Legend

| Status | Description |
|--------|-------------|
| `draft` | Work in progress — not yet reviewed or approved |
| `review` | Under review — awaiting feedback or sign-off |
| `approved` | Approved and ready for implementation |
| `implemented` | Design has been fully implemented |
| `deprecated` | No longer reflects the current system |
| `superseded` | Replaced by a newer HLD (reference the superseding HLD) |
| `template` | Placeholder — not a real design document |
